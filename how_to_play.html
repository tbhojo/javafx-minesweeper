How to Play Minesweeper  

 

Overview

Minesweeper is a game of logic and probability. The game begins with a grid of cells, some of which contain hidden mines. The goal of the game is to reveal all the cells that are safe, by clicking on them. A cell is safe if it doesn't contain a mine. Once the user has revealed all the safe cells, the game is won!

Types of Cells

  The game begins with unrevealed cells.

  If you click an unrevealed cell, you may uncover a number cell. Number cells range from 1 to 8 and represent the number of neighboring cells containing mines. 

  If you suspect that a cell contains a mine, then you can flag it by right-clicking the cell.
